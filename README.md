# Fantasy City Generator

Fantasy City Generator is a Go console application that creates fictional city names inspired by different fantasy cultures.

Each culture uses its own collection of prefixes and suffixes, producing names suitable for RPG campaigns, novels or world-building.

---

## Supported Cultures

- Elven
- Dwarven
- Nordic
- Desert Kingdom
- Imperial

---

## Features

- Random city generation
- Multiple cultural styles
- Generation statistics
- Clean modular structure
- Easy to extend

---

## Example

```
Selected Culture

Elven

Generated Cities

Aeloria
Lunareth
Sylanor
Elenmoor
Faelwyn

Statistics

Generated: 5
Culture: Elven
```

Run

```bash
go run .
```
